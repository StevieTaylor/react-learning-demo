# React Studying

记录学习和使用React的过程的项目


## 如何启动

使用yarn安装项目react-studying

```bash
  yarn install react-studying
  cd react-studying
```
    
## 如何部署

部署该项目, 需要运行

```bash
  yarn build
```

  
## 相关文档

[React官网](https://react.docschina.org/)
